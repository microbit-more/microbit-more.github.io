# Microbit More

Microbit More は教育向けプログラミング環境 Scratch (スクラッチ)で micro:bit (マイクロビット)を使えるようにする拡張機能です。Scratch に付属する micro:bit 拡張よりも高機能で、micro:bit に備わっているセンサーや LED などの機能をすべて利用できます。
